📚 Flashcard App 

A simple and interactive Flashcard Web Application built using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript.
This project allows users to create, manage, and review flashcards for learning and revision.

🚀 Features
➕ Add new flashcards (Question & Answer)
📖 View all flashcards
✏️ Edit existing flashcards
🗑 Delete flashcards
🔐 Basic Login Page (Frontend Authentication)
🎨 Responsive and colorful UI
🛠 Tech Stack
🔹 Backend
Node.js
Express.js
MongoDB (Mongoose)
🔹 Frontend
HTML
CSS
JavaScript

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/your-username/flashcard-app.git
cd flashcard-app
2️⃣ Install backend dependencies
cd backend
npm install
3️⃣ Run backend server
node server.js

Server will run on:

http://localhost:5000
4️⃣ Run frontend

Go to frontend folder and open:

login.html
