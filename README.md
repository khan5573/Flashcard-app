📚 Flashcard App

A simple and interactive Flashcard Application built using Node.js, Express, HTML, CSS, and JavaScript.
It allows users to create, view, and manage flashcards for study and revision.

🚀 Features
➕ Add new flashcards (Question & Answer)
📖 View all flashcards
💾 Data stored in JSON file (local storage)
⚡ Fast and lightweight UI
🎯 Useful for exam preparation
🛠️ Tech Stack
Backend: Node.js, Express
Frontend: HTML, CSS, JavaScript
Storage: JSON file
📁 Project Structure
FLASHCARD-APP/
│
├── data/
│   └── cards.json
│
├── public/
│   ├── index.html
│   ├── style.css
│   └── script.js
│

└── README.md
Install dependencies:
npm install
Run the server:
node server.js
Open in browser:
http://localhost:3000
